## 🏥 Sobre o Projeto

DocCoin é uma plataforma inovadora que revolutiona o acesso à saúde no Brasil através de tecnologia blockchain e tokenização. Conecta pacientes a profissionais de saúde qualificados, oferecendo telemedicina acessível, triagem inteligente com IA e um sistema de pagamento baseado em criptoativos.

**"Uber para Healthcare com a segurança de blockchain"**

---

## ✨ Principais Características

- 🎥 **Telemedicina** - Consultas seguras e criptografadas entre pacientes e médicos
- 🤖 **Triagem com IA** - Sistema inteligente de triagem automática para otimizar atendimentos
- 💰 **DocCoin Token** - Criptoativo nativo com tokenomics sustentável
- 🏢 **Planos de Saúde Flexíveis** - Assinaturas personalizáveis por especialidades e frequência
- 📍 **Mapa de Profissionais** - Localização inteligente de médicos e especialistas próximos
- 💳 **Carteira Digital** - Gerenciamento seguro de saldo em DocCoin e pagamentos

## 💼 Modelo de Negócio

### Tiers de Assinatura
- **Free** - Até 2 consultas/mês, triagem manual
- **Plus** - Até 8 consultas/mês + triagem IA + desconto em DocCoin (R$49/mês)
- **Premium** - Consultas ilimitadas + atendimento prioritário + cashback em tokens (R$199/mês)
- **Enterprise** - Para corporações, integração com RH e beneficiários ilimitados

### Tokenomics do DocCoin (DOC)
- **Supply Total** - 1 bilhão de tokens
- **Utilidade** - Pagamento de consultas, staking para desconto, governança
- **Distribuição** - 40% para usuários, 30% para médicos, 20% para desenvolvimento, 10% reserva
- **Modelo de Queima** - 2% das transações queimadas automaticamente

## 🎯 Como Rodar

### Pré-requisitos
- Node.js 16+ 
- npm 8+ ou yarn

### Instalação

```bash
# Clonar repositório
git clone https://github.com/doccoin/doccoin-health-app.git
cd doccoin-health-app

# Instalar dependências
npm install

# Configurar variáveis de ambiente
cp .env.example .env
# Editar .env com suas credenciais
```

### Desenvolvimento

```bash
# Iniciar servidor de desenvolvimento
npm run dev

# Acessar em http://localhost:5173
```

### Build para Produção

```bash
# Criar build otimizado
npm run build

# Preview da build
npm run preview
```
